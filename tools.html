Developed Tools

TSSr

An R/Bioconductor package designed for the comprehensive analysis of transcription start site (TSS) data. It provides a suite of tools for identification, visualization, and annotation of TSSs.

R Bioconductor TSS

autoBSgenome

A Python wrapper for BSgenome that simplifies the creation of BSgenome packages. It automates the process, allowing users to build custom genome packages as easily as filling out a questionnaire.

Python BSgenome Automation

DataColor

A tool for unveiling biological data relationships through distinctive color mapping. It helps researchers visualize complex datasets with intuitive and informative color schemes.

Visualization Color Mapping Genomics

PhyloForge

A comprehensive toolkit for unifying micro- and macro-evolutionary studies using genomic signals. It facilitates the construction of phylogenetic trees from various genomic data sources.

Phylogenetics Evolution Genomics
